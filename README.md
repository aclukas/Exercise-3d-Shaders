# Juicy Brick Breaker
Project for MSCH-C220

Description
A brick breaker game! Use A and D to move and space to launch the ball.

## Implementation
 * Sound effects made by me in Chiptone.
 * Music made by me in Soundtrap.
 * Randomized brick colors/the paddle changes color when hit. (The ball also does, but this isn't super noticeable)
 * Screen shake when hitting walls/bricks/paddle.
 * Bricks break on hit.
 * Bricks animate onto scene.
 * Bricks animate on destruction.
 * Particles when paddle is hit.
 * Comet trail following the ball.
 * Addition of eyes.
 * Eyes follow the ball.
 * Dynamic checkerboard background for main menu.
